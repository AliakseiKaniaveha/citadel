THE CITADEL PROJECT

Build. Fortify. Rest assured - it works!

Hi! I am a running instance of The Citadel Project. For the moment I can visualize all the classes I am constructed of and how they are interlinked - check it out.

I also have some plans for further growth. In the future I will be able to visualize classes of any application - all I'll need is just a jar file. Then I'd like to be able to show how differently various type of tests affects application classes graph.

But anyway that's not the point. The point is - I am an educational project which states that software properly guarded by automated tests is a considerably more sustainable and superior kind than software which lacks them. I can adopt any changes and have new features released to production safely and quickly, I'm always in a working state (no matter whether it is production or development) and anyone can start contributing in less than 5 minutes. My mission is to give it a taste of what it's like to develop software in that manner and my hope is that after that anyone tried it will consider this way to develop software as a de-facto standard and will bring this experience to their production projects. Briefly that's it, check out details on how to get there are.
